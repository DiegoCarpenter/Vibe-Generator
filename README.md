# Vibe-Generator

An AI-powered compliment and roast generator built with React and the Claude API. Dial in your intensity, pick your mode, and let it rip.

Features
Two modes — Compliment or Roast
Intensity slider — scale from wholesome grandma energy (1) to pure unhinged chaos (10)
Powered by Claude claude-sonnet-4-20250514
Setup
Clone the repo and install dependencies:

git clone https://github.com/DiegoCarpenter/Vibe-Generator.git
cd Vibe-Generator
npm install
Copy the env example and add your API key:

cp .env.example .env
Get a key at console.anthropic.com and paste it in:

VITE_ANTHROPIC_API_KEY=sk-ant-...
Start the dev server:

npm run dev
Open http://localhost:5173.
